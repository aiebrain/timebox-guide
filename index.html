1

GitHub 계정 만들기

이미 있으면 건너뛰세요
  1. 아래 링크를 클릭하세요
    https://github.com/signup
  2. 이메일 주소 입력 → 비밀번호 설정 → 닉네임(username) 입력
  3. 메일함에서 인증 코드 확인 후 입력
  4. 무료 플랜(Free) 선택 → 완료!
2

내 계정으로 복사하기 (Fork)

💡 "Fork"란? 원본 코드를 내 계정으로 가져오는 것입니다. GitHub에 로그인된 상태여야 해요!
  1. 아래 링크를 클릭하세요
    https://github.com/aiebrain/timebox-planner
  2. 페이지 오른쪽 상단 [Fork 🍴] 버튼 클릭
  3. [Create fork] 버튼 클릭
  4. 주소창이 github.com/내닉네임/timebox-planner 로 바뀌면 성공! ✅
3

Vercel로 배포하기

💡 "Vercel"이란? 코드를 인터넷에 올려주는 무료 서비스입니다.
  1. 아래 링크를 클릭하세요
    https://vercel.com/signup
  2. "Continue with GitHub" 버튼 클릭 (방금 만든 GitHub 계정으로 로그인)
  3. 상단 "Add New → Project" 클릭
  4. 목록에서 timebox-planner 찾아 [Import] 클릭
    ⚠️ 목록에 없으면 ② 단계(Fork)를 먼저 해주세요!
  5. 아무것도 건드리지 말고 [Deploy] 버튼 클릭
  6. 1~2분 기다리기 (초록색 로그가 내려가는 건 정상이에요 😊)
  7. 🎉 "Congratulations!" 화면이 뜨면 완료!
🎉
내 전용 URL이 생겼어요!
timebox-planner-내닉네임.vercel.app
📌 이 주소를 북마크해두세요!

사용 방법

단계기능
1️⃣ 브레인 덤프머릿속 할 일을 전부 쏟아내기
2️⃣ 빅 3 선정오늘 반드시 끝낼 3가지 선택
3️⃣ 타임라인시간대별 작업 배치 (드래그&드롭)
4️⃣ 리뷰하루 마무리 회고 저장
