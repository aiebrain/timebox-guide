GitHub 계정 만들기

코드를 저장하는 곳입니다 (무료)

이미 있으면 스킵 →
  • 1
    아래 링크를 클릭해서 가입 페이지로 이동하세요
    🌐https://github.com/signup
  • 2
    이메일 주소 입력 → 비밀번호 설정 → 닉네임(username) 입력
  • 3
    메일함에서 인증 코드 확인 후 입력
    📧메일이 안 보이면 스팸함도 확인해보세요!
  • 4
    요금제 선택 화면에서 무료(Free) 선택 → 완료!
2

내 계정으로 복사하기 (Fork)

원본 코드를 내 GitHub에 복사합니다

💡Fork란? 원본은 그대로 두고 내 계정에 똑같은 사본을 만드는 것입니다. 반드시 GitHub에 로그인된 상태여야 해요!
  • 1
    아래 링크로 이동하세요
    📁https://github.com/aiebrain/timebox-planner
  • 2
    페이지 오른쪽 상단에서 Fork 버튼을 찾아 클릭하세요
    👆 이 버튼을 클릭!  🍴 Fork
  • 3
    팝업창에서 Create fork 버튼 클릭
  • 4
    주소창이 바뀌면 성공!
    ✅주소창에 github.com/내닉네임/timebox-planner 가 보이면 완료입니다.
3

Vercel로 인터넷에 올리기

코드를 실제 웹사이트로 만들어줍니다 (무료)

💡Vercel이란? 내 코드를 인터넷 주소가 있는 웹사이트로 만들어주는 무료 서비스입니다.
  • 1
    아래 링크로 이동하세요
    🌐https://vercel.com/signup
  • 2
    Continue with GitHub 버튼 클릭
    ⚠️이메일/비밀번호가 아니라 GitHub 버튼으로 로그인해야 연동이 됩니다!
  • 3
    상단 메뉴에서 Add New → Project 클릭
  • 4
    목록에서 timebox-planner 를 찾아 Import 클릭
    🚨목록에 없나요? 위의 ② 단계(Fork)를 먼저 해주세요!
  • 5
    설정을 건드리지 말고 그냥 Deploy 버튼 클릭!
    ⏳1~2분 기다리세요. 초록색 로그가 쭉 내려가는 건 정상입니다 😊
🎉

자주 묻는 질문

GitHub에 로그인이 안 된 상태예요. 오른쪽 상단 [Sign in] 버튼을 먼저 클릭하세요.
Import 목록에 timebox-planner가 없어요
Fork를 먼저 해야 합니다. ② 단계를 다시 진행해주세요.
배포 중 빨간 에러가 났어요
